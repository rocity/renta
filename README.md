# Another Rent App

Rent app with Next.js and DRF

# Setup

### Django

#### Install dependencies with Pipenv

```bash
$ pipenv install
```

#### Run the development server

```bash
$ pipenv run python renta/manage.py runserver
```

### Next.js

#### Install dependencies with NPM

```bash
$ cd assets
$ npm install
```

#### Run the development server

```bash
$ npm run dev
```
